# KI-Guide Installation

Willkommen! Diese Anleitung zeigt dir Schritt für Schritt, wie du den KI-Guide auf deinem Computer startest.

Der KI-Guide läuft komplett **offline** auf deinem Rechner. Nichts wird verschickt, keine Internetverbindung nötig.

---

## 🪟 Windows

### Installation

1. Doppelklick auf `KI-Guide_x.x.x_x64-setup.exe`
2. **Windows SmartScreen meldet sich:** Das ist normal bei nicht signierten Apps.
   - Klick auf **„Weitere Informationen"**
   - Dann **„Trotzdem ausführen"**
3. Installer-Anweisungen folgen (Installationspfad bestätigen)
4. Fertig — KI-Guide startet automatisch

### Starten

- Über das **Startmenü → KI-Guide** oder
- Doppelklick auf das Desktop-Icon

---

## 🍎 macOS

### Installation

1. Doppelklick auf `KI-Guide_x.x.x_x64.dmg`
2. **Lara-Icon** in den **Programme-Ordner** ziehen
3. DMG schließen

### Erststart (wichtig!)

Da die App noch nicht bei Apple notarisiert ist, blockt macOS sie beim ersten Doppelklick.

**So startest du sie:**

1. Öffne den **Programme**-Ordner im Finder
2. **Rechtsklick** auf „KI-Guide" → **„Öffnen"**
3. Im Dialog **„Öffnen"** bestätigen
4. Fertig — danach kannst du sie ganz normal per Doppelklick starten

Falls das Fenster „App ist beschädigt" erscheint, öffne das Terminal (CMD+Space → „Terminal" eingeben) und führe einmalig aus:

```bash
xattr -cr /Applications/KI-Guide.app
```

Dann nochmal Rechtsklick → Öffnen.

## ❓ Probleme?

- **Video läuft nicht:** Stelle sicher, dass du die neueste Version verwendest. Auf Mac brauchst du **macOS 10.15 (Catalina) oder neuer**.
- **Audio kommt nicht:** Klick einmal auf den ▶️-Button im Player (Browser-Autoplay-Regeln).
- **App startet nicht:** Schreib uns an [DEINE-MAIL@DOMAIN.de].
